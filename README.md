# DevOps Pipeline Health Dashboard 🚀

A real-time CI/CD pipeline monitoring dashboard that tracks build status, failures, and performance metrics.

## Tech Stack
- Node.js (Backend)
- React (Frontend)
- Jenkins / GitHub Actions (CI/CD)
- Docker
- Kubernetes
- Prometheus & Grafana

## Features
- Real-time pipeline tracking
- Failure alerts
- Metrics visualization
- DevOps observability
